<h4> "또바기" 는 코로나로 인해 해외여행이 제한된 현 상황에 맞춰 제작된 맞춤형 여행지 추천 웹사이트 입니다.</h4> <br>
또바기에서 사용자는 간단한 테스트를 통해 맞춤형 국내 여행지와 추천 여행 스타일을 제공받을 수 있으며, 커뮤니티 기능을 통해 여행 후기를 서로 공유할 수 있습니다.
